### **Prerequisites**

Before you can run the project, ensure you have the following installed:

1. **Node.js**

   - Download and install Node.js from [here](https://nodejs.org/).
   - Verify installation:
     ```bash
     node -v
     npm -v
     ```

---

### **Run the Project**

To run:

```bash
cd frontend
npm install
npm start
```
